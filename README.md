## 有记忆的 AI 外语老师

**Slogan：查过，不等于记住。**

很多人不是不会，而是记不住。

同一个单词查了三次、五次、十次；  
同一个表达今天学会了，下周又忘了；  
AI 每次都能回答你的问题，却永远不记得你已经问过。

传统词典帮助你查到答案，却无法帮助你记住答案。

**记了么（EverLingo）是一款有长期记忆的 AI 外语老师。**

它会在你查词、翻译、阅读和提问的过程中，自动记录学习场景、积累知识轨迹，并在未来最需要的时候主动帮助你复习和强化记忆。

> 你负责学习，小记负责记住。



## 产品定位

记了么不是： AI 词典 、AI 翻译工具 

记了么是：

> 一个能够持续积累用户知识记忆的 AI 学习伙伴。

每一次查询，都会成为未来学习的一部分。

类比为一个专为外语学习优化的 OpenClaw 龙虾🦞。养一个外教🐹。

## 目标用户

- 学生
- 外语学习者
- 外语环境工作者
- 海外生活用户
- 技术、商务等专业领域从业者
- 希望长期积累语言能力的人群



## 核心能力

### 1. 把每一次查询变成长期记忆

传统工具：

查询 → 得到答案 → 结束

记了么：

查询 → 得到答案 → 记录场景 → 建立记忆 → 自动复习 → 长期掌握

系统自动保存：

- 原始问题
- 文章上下文
- 网页内容
- 屏幕截图
- 用户备注
- 历史学习记录

并自动生成：

- 知识笔记
- 生词卡片
- 记忆锚点
- 学习档案

让每一次查询都成为未来学习资产的一部分。



### 2. 越用越懂你的 AI 老师

记了么会逐渐理解：

- 用户语言水平
- 职业背景
- 兴趣领域
- 常见错误模式
- 学习习惯

因此，同一个知识点会因人而异：

- 程序员获得技术场景解释
- 商务人士获得商务场景解释
- 学生获得考试场景解释

每个人看到的内容都更适合自己。



### 3. 自动发现遗忘并安排复习

系统会主动识别：

- 高频重复查询内容
- 容易遗忘的单词和表达
- 经常答错的知识点
- 长期未复习内容

并自动生成：

- 个性化复习计划
- 测验题
- 场景练习
- 强化记忆任务

帮助用户把短期记忆转化为长期记忆。



## 核心创新

大多数 AI 产品都在解决：

> 如何更快获得答案。

记了么解决的是：

> 如何让答案真正留下来。

传统 AI：

提问 → 回答 → 遗忘

记了么：

提问 → 回答 → 记录 → 理解 → 强化 → 掌握

**把查询行为本身变成学习资产。**



## 为什么用户会选择记了么

- 查过的不再白查。每一次查询都会沉淀为个人知识资产。
- 知识永远带着场景。不仅记住单词，更记住为什么会查它。
- AI 不会忘记你的学习历史。无需重复解释自己的水平和背景。
- 自动帮你复习。从被动学习变成主动掌握。

## Quick Start

### 源码运行


TUI:
```bash
uv run python -m everlingo.main
# or
uv run python -m everlingo.gateway.gateway --channel_stdio
```

微信:
```bash
uv run python -m everlingo.gateway.gateway --channel_wechat
```

#### 运行 Web 服务

两个终端：
终端 1 — 后端（FastAPI + uvicorn）
```bash
.venv/bin/python -m everlingo.gateway.gateway --channel_web
```
启动后监听 http://localhost:8000，提供 API 和静态文件。

终端 2 — 前端开发（Vite 热更新）
```bash
cd web && npm run dev
```
启动后监听 http://localhost:5173，/api/* 自动代理到后端 8000 端口。

生产部署只需 cd web && npm run build 构建一次，FastAPI 自动从 web/dist/ 提供前端文件。后端进程启动后，直接访问 http://localhost:8000 即可。
