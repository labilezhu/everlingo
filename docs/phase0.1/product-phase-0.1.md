
# 产品说明 - phase 0.1

产品中文名称：依娃外教 
产品英文名称：everlingo

一句话产品说明：一款个性化的 AI 外语学习助手

在学习外语 或 在外语环境工作时，我们时常要查询单词、术语。而很多时候，一个相同单词的单词因长时间遗忘需要被多次查询。如果能场景化地记下每一次查询，建立记忆锚点，如原题目，原文章段落，原网页，原电影或屏幕截图，再加入一些辅助记忆又合符用户文化背景的资料卡片。在之后用户重复查询，就可以帮助巩固记忆。



## 应用特性

### 功能

一个应用。它的目标是作为一个一对一的外语老师。它可以完成多个用户意图：
- **查单词**。记录用户查询过的单词。分析查询历史，制订专用词汇表和记录查询历史和频率。查词时用户可以记录单词出现的场景，可以是文字。LLM 对场景分析，记录。用以为用户回忆作引导。
- **翻译**。分析翻译历史，提取高频内容，句式，短语。

### 特性

- 单词
	- 单词应该提供对应用户文化背景的定制解释、目标语言的词源故事、生成辅助记忆的图形卡片

## 产品设计

### 界面
- 终端 TUI 界面的 Chatbot

### 用户个性初始化
`必须设置项`：
- `界面语言`
- `目标学习语言`。约束规则：不能和 对话语言 相同



### 语言的支持
界面语言和目标学习语言只能选择：
- 英语
- 简体中文

### Chatbot

分析判断用户意图，然后作出响应。

用户意图包括：
1. `引导初始配置` 完成 用户个性初始化 中的 `必须设置项`
2. `查单词`
3. `翻译`

#### 用户意图分析

- 如果用户个性初始化的`必须设置`未设置，需要先`引导初始配置`。

- 如果输入的用户消息以`目标学习语言`文本开始，且中间没有 `界面语言` 文字类文本：
  - 如果是一个词（英语的单词或中文的词）则判断用户意图为 `查单词`
  - 如果是多个词（英语的单词或中文的词）则判断用户意图为 `翻译`
