# 产品设计

### 界面
- Chatbot 
  - TUI
  - 可集成到微信（基于 微信 iLink Bot ）
- Self-hosted Web
- Chrome Extension 网页词典，类似 Google Translate
- Apple iOS 的原生词典，支持 iOS 翻译和词典功能的整合

### 用户个性初始化
`必须设置项`：
- `界面语言`。约束规则：不能和 `目标学习语言` 相同
- `目标学习语言`。约束规则：不能和 `界面语言` 相同

### Chatbot

分析判断用户意图，然后作出响应。

用户意图包括：
1. `引导初始配置` 完成 用户个性初始化 中的 `必须设置项`
2. `查单词`
3. `翻译`

#### 用户意图分析

- 如果用户个性初始化的`必须设置`未设置，需要先`引导初始配置`。

- 如果输入的用户消息以`目标学习语言`文本开始，且中间没有 `界面语言` 文字类文本：
  - 如果是一个词（英语的单词或中文的词）则判断用户意图为 `查单词`
  - 如果是多个词（英语的单词或中文的词）则判断用户意图为 `翻译`
