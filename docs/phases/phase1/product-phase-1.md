
# 产品说明 - phase 1 - DEMO

这是产品的 Demo 版本。只选择 [PRODUCT.md](/PRODUCT.md) 中的以下部分来实现。目标是一个简单的可运行，可 Demo 版本。


在学习外语 或 在外语环境工作时，我们时常要查询单词、术语。而很多时候，一个相同单词的单词因长时间遗忘需要被多次查询。如果能场景化地记下每一次查询，建立记忆锚点，如原题目，原文章段落，原网页，原电影或屏幕截图，再加入一些辅助记忆又合符用户文化背景的资料文本。在之后用户重复查询，就可以帮助巩固记忆。


## 应用特性

### 功能

一个应用。它的目标是作为一个一对一的外语老师。它充当多个角色(Agent)：
- **词典老师**。记录用户查询过的单词。分析查询历史，制订专用词汇表和记录查询历史和频率。查词时用户可以记录单词出现的场景，可以是文字。LLM 对场景分析，记录。用以为用户回忆作引导。
- **翻译老师**。分析翻译历史，提取高频内容，句式，短语。

### 特性

- 单词
	- 单词应该提供对应用户文化背景的定制解释、词源故事
